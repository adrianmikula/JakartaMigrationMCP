## [1.0.0] - 2026-01-31

### Added

- Initial release of Jakarta Migration MCP Server.
- MCP tools: Jakarta readiness analysis, blocker detection, version recommendations, migration plans, impact analysis, runtime verification.
- Node.js wrapper for npm/npx: downloads and runs the Java JAR from GitHub Releases.
- Support for stdio (local) and SSE/Streamable HTTP (e.g. Apify) transports.
- BSL 1.1 license; commercial use requires a separate license.
