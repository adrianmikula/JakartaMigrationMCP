# Premium Documentation

This directory contains documentation for the **premium modules** of the Jakarta Migration project.

## Overview

Premium modules provide advanced features beyond the community edition. They are **proprietary** and require a commercial license to use.

## Premium Features

| Module | Description | Key Features |
|--------|-------------|--------------|
| `premium-core-engine` | Advanced migration features | Enhanced AI analysis, priority support, custom recipes |
| `premium-mcp-server` | Premium MCP server | Higher rate limits, dedicated endpoints, priority queue |
| `premium-intellij-plugin` | Premium IDE integration | Advanced UI, real-time sync, team collaboration |
